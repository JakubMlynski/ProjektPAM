Magic The Gathering - Aplikacja Flutter

Aplikacja mobilna służąca do przeglądania kart oraz zestawów (setów) z gry Magic The Gathering. Umożliwia wyszukiwanie kart, przeglądanie ich szczegółów oraz działa w trybie offline (zachowuje ostatnio pobrane dane).

Wykorzystane API
Dane pobierane są z publicznego API Magic The Gathering:
https://docs.magicthegathering.io/

Funkcjonalności
- Przeglądanie listy kart z paginacją (dładowywanie kolejnych stron).
- Wyszukiwarka kart po nazwie.
- Przeglądanie listy zestawów (Setów).
- Szczegółowy widok karty (grafika, opis, mana, typ).
- Obsługa trybu offline (zapisywanie danych lokalnie).
- Obsługa błędów i stanów ładowania.

Instrukcja uruchomienia (windows)

cd (lokalizacja aplikacji)
flutter pub get
flutter run
